## 注意事项

### V8 API 警告
在重新编译 better-sqlite3 时出现了一些 V8 API 弃用警告：
```
warning C4996: 'v8::CopyablePersistentTraits<T>': Use v8::Global instead
```
这些警告是正常的，不会影响功能，是由于 better-sqlite3 使用了一些即将弃用的 V8 API。
