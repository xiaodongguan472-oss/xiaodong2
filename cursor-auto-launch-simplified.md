# Cursor自动启动功能 (简化版)

## 功能概述

在刷新Cursor账号成功后，系统会自动：

1. **检查Cursor运行状态** - 判断Cursor是否已经在运行
2. **自动启动Cursor** - 如果Cursor未运行，则自动启动

## 实现的功能

### 后端IPC处理函数

#### `check-cursor-running`
- **功能**: 检查Cursor是否正在运行
- **实现**: 使用系统命令检查进程
- **返回**: boolean值

#### `launch-cursor`
- **功能**: 启动Cursor应用程序
- **实现**: 跨平台启动逻辑
- **返回**: 成功/失败状态

### 前端逻辑

在账号切换成功后：

1. **检查运行状态**
   ```javascript
   const isRunning = await ipcRenderer.invoke('check-cursor-running');
   ```

2. **条件启动**
   - 如果未运行：启动Cursor
   - 如果已运行：显示提示

### 用户体验

- **智能启动**: 只在Cursor未运行时才启动
- **友好提示**: 根据不同情况显示相应消息
- **跨平台支持**: 支持Windows、macOS和Linux

### 消息提示

- Cursor未运行且启动成功：`续杯成功！Cursor已自动启动`
- Cursor已运行：`续杯成功！Cursor已在运行中`
- 启动失败：`续杯成功！但启动Cursor失败，请手动启动`

## 使用流程

1. 用户点击"刷新Cursor"按钮
2. 系统验证授权码并切换账号
3. 自动检查Cursor运行状态
4. 如果未运行则启动Cursor
5. 显示相应的成功消息

## 技术特点

- **智能判断**: 避免重复启动
- **容错性**: 启动失败不影响主要功能
- **跨平台**: 支持所有主流操作系统

这个简化版本专注于自动启动功能，移除了AI模型设置部分，更加简洁实用。
